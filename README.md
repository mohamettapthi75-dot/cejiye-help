# cejiye-help 🇸🇴

> Adduunka koowaad ee tool coding ah oo Af Somali ku hadla.

Marka aad khalad coding ku hesho — cejiye-help Somali kugu sharxaa
sababta iyo xalka. Termux-ka ku shaqeeya, internet ma u baahna.

## Tusaale

cejiye git push

⚠️  KHALAD: Push-ku fashilmay
📋 Sabab:   Koodkaagu ka dambeeyaa server-ka
✅ Xal:     Marka hore `git pull` qor, kadibna `git push` isku day

## Sida Loo Rakibo

git clone https://github.com/mohamettapthi75-dot/cejiye-help.git
cd cejiye-help
bash install.sh

## Amarrada

| Amarka | Waxa Sameeya |
|--------|-------------|
| `cejiye list` | Errors-ka oo dhan muuji |
| `cejiye add` | Error cusub ku dar |
| `cejiye history` | Taariikhda errors-ka eeg |
| `cejiye stats` | Errors-ka ugu badan muuji |

## Errors-ka Taageersan

- 🔧 Git — push, pull, merge
- 🐍 Python — syntax, modules
- 📦 NPM — install, modules
- 📱 Termux — storage, packages

## Sababta Loo Sameeyay

Coding baranaya oo Somali ah ma heli karaan caawimo luqadooda.
cejiye-help waxay xallisaa dhibaatadan — errors-ka Somali kugu
sharxaa si aad u fahanto, si aad u barato.
